 # Educational Keylogger

This is a simple keylogger created for educational purposes only. It logs keystrokes to a file called `details.txt` with timestamps.

## Installation

1. Make sure you have Python installed on your system
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the keylogger:
   ```
   python keylogger.py
   ```
2. The program will start logging keystrokes to `details.txt`
3. Press the 'esc' key to stop the keylogger

## Important Notes

- This keylogger is for educational purposes only
- Never use this for malicious purposes
- Always get proper authorization before using on any system
- The logs are saved in `details.txt` in the same directory as the script

## Features

- Logs keystrokes with timestamps
- Easy to start and stop (press 'esc' to stop)
- Saves logs to a text file
- Includes error handling
